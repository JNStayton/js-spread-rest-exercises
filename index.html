Spread and Rest Practice

    Practice in the Console!

  • filterOutOdds() accepts a variable amount of numbers, filters out the odds, and returns an array.
  • findMin() accepts a variable amount of numbers, returns the smallest argument.
  • mergeObjects(obj1, obj2) accepts two objects and returns a new object which contains all the keys and values of the first and second object.
  • doubleAndReturnArgs(array, arg1, arg2, arg3) accepts an array and a variable amount of arguments. Returns a new array with the original array values, and all of the additional arguments doubled.
  • removeRandom(array) accepts an array; removes a random element in an array and returns a new array without that item.
  • extend(array1, array2) accepts two arrays, returns a new array with every item from the first and second array.
  • addKeyValue(obj, key, val) accepts an object, a key, and a value; returns a new object with all the keys and values from the object and a new key/value pair.
  • removeKey(obj, key) accepts an object and a key; returns a new object with the given key removed.
  • combine(obj1, obj2) accepts two objects; returns a new object with every key/value pair from both objects.
  • update(obj, key, val) accepts an object, a key, and a value; returns a new object with a modified key and value
